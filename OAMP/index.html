This page was developed to verify, after a successful installation, the correct functioning of httpd-apache in OpenBSD systems.


If you accessed this page, the apache web server is installed and running on your system.

This page was customizing following the code of the index files of several linux distributions, and adapted to OpenBSD systems by victxrlarixs

Documentation

The Apache manual is avaible with distribution as part of the man file set, and installed in /usr/share/doc/httpd.

Especially read the SSL documentation carefully within the manual, in addition to the ssl and manual mages.
